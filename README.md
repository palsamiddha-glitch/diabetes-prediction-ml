# 🧬 Diabetes Prediction using Machine Learning

## 📌 Overview
This project predicts whether a person is diabetic or not based on medical parameters using Machine Learning.

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Google Colab

## 🤖 Algorithm Used
- K-Nearest Neighbors (KNN)

## 📊 Features Used
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

## 🎯 Objective
To build a predictive model that can classify patients as diabetic or non-diabetic using health data.

## 📈 Result
- Achieved approximately 65–75% accuracy on test data
- Successfully implemented real-time prediction using user input

## 🚀 How to Run
1. Open the notebook (`.ipynb`) in Google Colab
2. Run all cells
3. Enter patient details when prompted
4. Get prediction result (Diabetic / Not Diabetic)

## 📁 Dataset
Dataset sourced from Kaggle (Pima Indians Diabetes Dataset)
